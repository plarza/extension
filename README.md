# Plarza Extension

Passively scans web pages for URLs and submits them to the Plarza worker API for content ingestion

## Prerequisites
- A userscript manager ([Tampermonkey](https://www.tampermonkey.net/), [Violentmonkey](https://violentmonkey.github.io/), etc.)

## Installation
2. Install the userscript from the URL [https://raw.githubusercontent.com/plarza/extension/refs/heads/main/dist/plarza.user.js](https://raw.githubusercontent.com/plarza/extension/refs/heads/main/dist/plarza.user.js)
3. On first page load, you'll be prompted for your Plarza API key
